Display Random Products

== Description ==

This Plugin provides a Shortcode, to display three Random Products per Call.

It uses Material Design Bootstrap CSS Code - No complications with other CSS Kits are known.

You can use it, either through the Backend and choose multiple Categories, or you can use it to display Random Products of a defined Category.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/product-list-box` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Shortcode Produktebox screen to configure the plugin
4. Use the Shortcode [product_listing] in every Post / Page / Custom Post Type you want
